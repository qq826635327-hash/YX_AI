# 13 角色

## 1. 路由

`/projects/:id/characters` → `<Characters />`

## 2. 布局

```
左栏：剧集/项目（无）  中栏：角色网格  右栏：选中后角色详情
```

## 3. 角色网格

- 卡片：形象缩略图 + 名称 + 性别 / 年龄 / 标签
- 操作：选中（多选） / 打开详情 / 删除 / 批量生成形象 / 导出
- 顶部工具栏：
  - 搜索
  - 筛选：性别、年龄、标签
  - 排序：最近编辑 / 名称
  - 视图：网格 / 列表
  - "新建角色"按钮
  - "从剧本导入"按钮（解析 剧本.json 里的 characters 字段）

## 4. 角色详情

字段：

- 基础：名称、性别、年龄、简介、标签
- 形象：主图、参考图集（多张）
- 提示词：`appearancePrompt`（用于生图时锁定一致性）
- 关联：linkedShots / linkedScenes / linkedProps
- 历史：生成结果时间线
- 操作：保存、删除、复制为新角色、批量应用到关联分镜

## 5. 形象生成

点"生成形象"→ 弹"生成对话框"：

```
模型     [图片模型下拉]
风格     [风格预设 + 自定义]
比例     [1:1 3:4 4:3 9:16 16:9]
参考图   [拖入/选择]
提示词   [自动 = appearancePrompt + 风格 + 比例]
数量     [1 2 3 4]
确认     [生成]
```

提交 → 走"出站" → 结果写入 `角色/<id>/生成/<时间戳>.png` + 嵌入 mediaMeta → 弹出灯箱 → 用户可"设为形象 / 加入参考图 / 重新生成"。

## 6. 提示词助手

点"AI 改写提示词"→ 一次文本模型调用：

```
prompt = 模板("角色外貌扩写", { name, summary, currentPrompt, style })
→ 返回更丰富的外貌描述（中文）
→ 写回 appearancePrompt
```

内置预设：

- 默认
- 二次元动漫画风
- 3D 渲染风格
- 水墨插画风格

## 7. 参考图

- 拖入 / 上传图片
- 每张图可标注角色部位（脸 / 服装 / 全身 / 表情 / 其它）
- 生图时选择几张作为参考，传给 `outbound:request.images`

## 8. 关键 IPC

- `fs:*`（读写 JSON / PNG）
- `outbound:request`（生图、改写提示词）
- `mediaMeta:embed`
- `clipboard:copy-image`（右键"复制图片"）
- `dialog:save-file`（导出）

## 9. 状态

- 角色 JSON 永远有 `id`
- 删除角色：先确认 → 解除所有分镜的关联 → 删除文件
- 角色被剧本引用但未建角色卡："未绑定"状态
