# 20 设置 · 供应商与模型

## 1. 入口

`/settings/providers` 与 `/settings/models`、`/settings/prompts`

## 2. 供应商管理

```
供应商列表
├── + 新增供应商
├── 火山方舟 (volc-ark)         [已配置] [测通] [编辑] [删除]
├── 千问 (openai-compatible)    [未配置] [编辑] [删除]
├── DeepSeek (openai-compatible) [已配置] [测通] [编辑] [删除]
├── Claude Agent                [独立 SDK] [连接] [管理 Skill]
├── 自定义 OpenAI 兼容 × N      ...
└── ComfyUI                     [本地] [编辑]
```

新增供应商表单：

- 名称、类型（openai-compatible / volc-ark / claude-agent / comfy / custom）
- baseUrl（可选）
- API Key（可选，存到 userData/.enc 加密）
- 区域（火山方舟必填）
- 启用 / 禁用
- 模型列表（先选 kind：text / image / video / audio，再填 model id）

## 3. 凭证存储

- 走 `safeStorage`（Windows DPAPI / macOS Keychain / Linux libsecret）
- 存到 `userData/.enc`
- 主进程用 `safeStorage.encryptString(plain)` → 落盘
- 读：解密后只放在主进程内存，从不返回 renderer 明文（除非 `revealCredentials` 用户主动触发）

## 4. 测通

- OpenAI 兼容：发 `GET /v1/models` 或 `POST /v1/chat/completions` 一条空请求
- 火山方舟：发 `ListAssets`（pageSize=1）
- Claude Agent：发一个 `claudeAgent:test-connection` 调用
- ComfyUI：发 `GET /system_stats`

## 5. 模型配置

- 模型表：id、name、kind（text/image/video/audio）、defaultFor（默认用于）、params、是否启用
- 每个项目可覆盖：`project.json.settings.imageModel`

## 6. 提示词模板

### 6.1 列表

按 scope 过滤：

- script：剧本解析、分镜拆解
- shot：首帧 / 尾帧 / 视频 / 音频 / 改写
- character：外貌扩写、形象生成
- scene：场景概念图
- prop：道具图
- custom：用户自建

### 6.2 编辑器

- 主文本区（Markdown）
- 占位符下拉：插入 `{{xxx}}` 变量
- 变量高亮
- "测试运行"按钮：用当前选中项目数据填入变量跑一次，结果预览

### 6.3 导入 / 导出

- 导出：`{version, templates: [...]}` JSON
- 导入：覆盖 / 合并 / 跳过

## 7. 关键 IPC

- `outbound:register-secrets` / `clear-secrets` / `clear-all-secrets`
- `outbound:set-defaults`
- `volc:save-credentials` / `clear-credentials` / `test-credentials` / `get-summary` / `reveal-credentials`
- `claudeAgent:test-connection` / `list-custom-skills` / `save-custom-skill` / `delete-custom-skill` / `get-custom-skills-dir` / `reveal-custom-skills`

## 8. 状态持久化

供应商 / 模型 / 提示词模板在 `userData/data/settings.json`：

```json
{
  "providers": [...],
  "models": [...],
  "templates": [...],
  "defaultProjectDirectory": "...",
  "outboundDefaults": { "timeoutMs": 120000, "retry": 1 }
}
```
