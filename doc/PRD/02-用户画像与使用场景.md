## 2. 核心使用场景（Use Cases）

### UC-01：从零开始一部短剧（端到端主流程）

**主角**：小北  
**前置条件**：已启动前后端，浏览器打开 `http://127.0.0.1:5173/`  
**目标**：将一段原创剧本变成包含角色 / 场景 / 分镜的短剧项目

**主流程**：
1. 进入首页 → 点击"新建项目" → 填写项目名"都市夜归人"、描述 → 创建
2. 自动跳转到项目总览 → 看到 6 个快捷卡片（剧本 / 角色 / 场景 / 道具 / 剧集 / 任务）
3. 进入剧本页 → 粘贴 5000 字剧本 → 选择画风"realistic" → 点击"AI 解析"
4. 等待 3 分钟，LogViewer 显示解析进度（3 阶段：实体提取 → 章节切分 → 分镜拆解）
5. 解析完成 → 自动跳转角色页 → 看到 8 个角色卡片，每人有 name / gender / age
6. 选中主角"陆远" → 打开 GenerateDialog → 选 Agnes Image 2.1 Flash → 生成主图
7. 切换到剧集页 → 看到 3 集，每集 10-15 个分镜 → 每个分镜含 7 段 Prompt
8. 为第 1 集第 1 个分镜添加角色引用（陆远、女主角苏晚）→ 生成首帧 → 生成视频
9. 任务中心看到所有任务进度 → WebSocket 实时更新

**异常流程**：
- 解析失败 → 状态变 `failed` → LogViewer 红点闪烁 → 点击复制 JSON 日志 → 重新解析
- 生成任务失败 → 任务卡片显示错误 → 点击"重试"按钮
- 磁盘目录异常 → 点击素材画廊"同步"按钮 → 双向修复

### UC-02：批量生成角色主图

**主角**：小北  
**目标**：为项目内所有未生成主图的角色批量生成

**主流程**：
1. 进入角色页 → 勾选 8 个 `gen_status = pending` 的角色
2. 点击"批量生成" → GenerateDialog 进入 batch 模式
3. 选 Provider = Agnes、Model = agnes-image-2.1-flash、Size = 1024x1024
4. 点击"提交" → 创建 8 个 GenerationTask，状态 pending
5. 任务中心看到 8 个任务并行执行（并发限制 4）
6. 约 2 分钟后全部 succeeded → 角色卡片自动刷新显示主图

### UC-03：配置新 Provider

**主角**：阿远  
**目标**：添加 SenseNova Provider，并配置其 Flash 模型

**主流程**：
1. 进入设置 → API 供应商 → 点击"新增"
2. 填写：name=SenseNova、kind=openai、base_url=https://token.sensenova.cn/v1、API Key（加密存储）
3. 点击"测试连接" → 成功
4. 切换到"模型配置"标签 → 添加模型 SenseNova-6.7-Flash-Lite
5. 配置 param_specs（size、n、seed）和 capabilities（text_reasoning）
6. 保存 → 该 Provider 即可在 GenerateDialog 中被选用

### UC-04：管理项目资产

**主角**：小北  
**目标**：整理角色"陆远"的所有生成图，设置默认主图

**主流程**：
1. 角色页点击"陆远"卡片 → 进入详情页
2. 看到 AssetGallery：12 张生成图，按创建时间倒序
3. 点击某张图的星标按钮 → 设为 `image_asset_id`（主图）
4. 上传一张参考图（PNG 300KB）→ 自动注册到 DB + 落入 `角色/陆远/images/` 目录
5. 点击"同步"按钮 → 扫描磁盘，发现一张未注册图 → 自动补登

## 3. 用户旅程地图（User Journey）

```
[认知]          [安装/启动]      [创建项目]       [AI 解析]       [资产生成]       [发布]
  ↓                ↓                ↓                ↓                ↓              ↓
GitHub/友链    跑 start-dev     首页新建项目    粘贴剧本+解析    生成主图/视频    导出分镜素材
                开浏览器        填项目信息      看 LogViewer     任务中心跟踪    （Phase 4）
                                                等待 3 分钟      画廊筛选
                                                                 重试失败任务
```

**情绪曲线**：期待 → 轻微焦虑（安装）→ 兴奋（看到项目结构）→ 等待焦虑（解析）→ 满足（看到角色）→ 兴奋（看到生成图）→ 满足（导出）。
