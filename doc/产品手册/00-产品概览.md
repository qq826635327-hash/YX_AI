# 00. 产品概览

## 1. 产品定位

AI Drama Studio 是面向 AI 短剧/剧集生产的桌面级 Web 工作台。整个工作流从剧本出发，逐步产出**角色、场景、道具、剧集分镜**及其对应的图片与视频。

### 1.1 核心目标
让创作者在一个统一的 Web 界面内：
1. 管理项目级资产（角色/场景/道具/分镜）
2. 调用 AI 模型生成图片与视频
3. 在分镜层组织多模态资产
4. 跨项目跟踪所有生成任务

### 1.2 适用场景
- 短剧 / 网剧前置可视化
- 漫画 / 短片脚本图化
- AI 影像工作流调试
- 教学演示

## 2. 信息架构

### 2.1 顶层模块
应用分为三大模块（通过 TopBar 切换）：

| 模块 | 入口 | 用途 |
|------|------|------|
| **项目** | TopBar 项目下拉 | 进入具体项目工作台 |
| **任务** | TopBar 任务图标 | 查看所有生成任务 |
| **配置** | TopBar 齿轮图标 | 系统级配置 |

### 2.2 侧边栏菜单

**项目内容**（进入项目后）：
- 剧本
- 角色
- 场景
- 道具
- 剧集结构

**任务中心**（独立模块）：
- 全部
- 进行中
- 已完成
- 失败
- 已取消

**配置中心**（独立模块）：
- API 供应商
- 插件扩展
- 模型配置
- 提示词模版
- ComfyUI 服务器
- ComfyUI 工作流

## 3. 工作流程（Happy Path）

```
1. 新建项目（[首页项目列表](./02-首页项目列表.md)）
2. 录入完整剧本（[剧本](./04-剧本.md)）
3. AI 解析 → 自动生成 角色/场景/道具/剧集
4. 角色/场景/道具 各自生成"主图"（[角色](./05-角色.md)、[场景](./06-场景.md)、[道具](./07-道具.md)）
5. 进入剧集结构，逐分镜补全描述（[剧集结构](./08-剧集结构.md)）
6. 分镜级生成首帧/尾帧/视频
7. 在任务中心统一跟踪所有生成任务（[任务中心](./09-任务中心.md)）
```
