# 02. 首页 / 项目列表

**路由**：`/`
**对应文件**：`pages/HomePage.tsx`

## 1. 功能描述

首页是应用的入口，提供项目列表与项目创建功能。

## 2. 页面行为

- **自动跳转**：如果存在项目，自动跳转到最近操作的项目（按 `updated_at` 倒序）
- **URL 参数**：`?create=1` 强制弹出新建项目对话框
- **空状态**：无项目时显示"还没有项目"引导页

## 3. 页面元素

- 页面标题：`AI Drama Studio`
- 副标题：`选择一个项目开始创作`
- **新建项目按钮**（在空状态或对话框中）

## 4. 新建项目对话框（CreateProjectDialog）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | text | ✅ | 唯一 |
| 项目描述 | textarea | ❌ | 多行 |

**提交后**：跳转到新项目的 [项目总览](./03-项目总览.md) 页

## 5. 项目卡片（ProjectCard）

项目卡片在首页和项目总览中复用，显示：

- 项目封面/占位图标
- 项目名称
- 项目描述
- **统计信息**：
  - 角色数量
  - 场景数量
  - 道具数量
  - 剧集数量（含分镜总数）
- 创建时间 / 更新时间
- "打开项目" 按钮

## 6. 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects` | 项目列表（支持分页、状态过滤、关键词搜索） |
| POST | `/api/projects` | 新建项目（自动创建项目根目录） |
| GET | `/api/projects/{id}` | 项目详情（含统计数据） |
| PATCH | `/api/projects/{id}` | 更新项目 |
| DELETE | `/api/projects/{id}?delete_files=...` | 删除项目 |

## 7. 数据模型（Project）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | UUID |
| name | str | 名称 |
| description | str | 描述 |
| cover_image | str | 封面图路径 |
| status | str | active / archived |
| root_path | str | 项目根目录（自动生成） |
| character_count | int | 角色数（统计缓存） |
| scene_count | int | 场景数（统计缓存） |
| prop_count | int | 道具数（统计缓存） |
| episode_count | int | 剧集数（统计缓存） |
| shot_count | int | 分镜数（统计缓存） |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### curl 快速验证

```powershell
# 1. 创建项目
curl -X POST http://127.0.0.1:8000/api/projects -H "Content-Type: application/json" -d "{\"name\": \"AI测试项目\", \"description\": \"自动测试\"}"
# 期望: 200, 返回含 id/name/root_path 的 JSON

# 2. 查询列表（分页）
curl "http://127.0.0.1:8000/api/projects?page=1&page_size=10"
# 期望: 200, 返回分页列表

# 3. 同名创建（冲突）
curl -X POST http://127.0.0.1:8000/api/projects -H "Content-Type: application/json" -d "{\"name\": \"AI测试项目\"}"
# 期望: 409 或 422

# 4. 删除项目（级联清理）
curl -X DELETE "http://127.0.0.1:8000/api/projects/{pid}?delete_files=true"
# 期望: 200, 所有子数据和磁盘文件清理
```
