# 03. 项目总览

**路由**：`/projects/{projectId}`
**对应文件**：`pages/ProjectDetailPage.tsx`

## 1. 功能描述

项目工作台的总览页面，提供项目全局信息查看与快速入口。

## 2. 页面元素

### 2.1 项目头部
- 项目名
- 项目描述
- 状态徽章（active / archived）
- 编辑/归档按钮

### 2.2 统计卡片

| 卡片 | 内容 |
|------|------|
| 角色 | 总数 + 分类分布（主角/配角/群演） |
| 场景 | 总数 |
| 道具 | 总数 |
| 剧集 | 总数 + 分镜总数 |
| 任务 | 各状态分布 |

### 2.3 快捷入口卡片

| 卡片 | 显示 |
|------|------|
| 剧本 | 带解析状态徽章 |
| 角色 | 带主图就绪状态 |
| 场景 | 带主图就绪状态 |
| 道具 | 带主图就绪状态 |
| 剧集 | 带分镜数量 |

### 2.4 最近活动
最近更新的实体列表（角色/场景/道具/分镜混合）

## 3. 交互

- 点击统计卡片 → 跳转到对应模块
- 顶栏的"项目"按钮下拉可快速切换项目
- "打开项目" 按钮跳转到默认工作区

## 4. 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects/{id}` | 项目详情（含统计数据） |
| PATCH | `/api/projects/{id}` | 更新项目 |
| POST | `/api/projects/{id}/archive` | 归档项目（待实现） |
