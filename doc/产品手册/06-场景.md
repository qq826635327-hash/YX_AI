# 06. 场景

**路由**：
- 列表：`/projects/{projectId}/scenes`
- 详情：`/projects/{projectId}/scenes/{sceneId}`

**对应文件**：
- `pages/ScenesPage.tsx`
- `pages/SceneDetailPage.tsx`
- 复用组件：[通用组件（EntityCard + EntityEditDialog）](./16-通用组件.md)

## 1. 功能描述

场景模块管理项目中的所有"地点/环境"资产。每个场景是剧集发生的物理或概念空间。

## 2. 与角色模块的差异

- **没有 Tabs 分类**（场景不分类）
- **编辑字段少 1 个**（无 `char_type`）
- 其余功能（卡片、编辑框、详情、生成、删除）与角色完全一致

## 3. 场景卡片

- 场景主图
- 场景名
- 描述 / 设定
- 操作按钮（生成/编辑/删除）

## 4. 场景编辑字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | text | ✅ | 项目内唯一 |
| 描述 | textarea | ❌ | 多行 |
| 设定 | textarea | ❌ | 用于生成 |

## 5. 场景详情页

同 [角色详情页结构](./05-角色.md#4-角色详情页)，但：
- 顶部图标为地图（Map）
- 无分类徽章
- 关联分镜：显示使用了此场景的分镜

## 6. 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects/{id}/scenes` | 列表 |
| POST | `/api/projects/{id}/scenes` | 新建 |
| GET | `/api/projects/{id}/scenes/{sid}` | 详情 |
| PATCH | `/api/projects/{id}/scenes/{sid}` | 更新 |
| DELETE | `/api/projects/{id}/scenes/{sid}` | 删除 |

## 7. 数据模型（Scene）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | UUID |
| project_id | str | 所属项目 |
| name | str | 名称 |
| description | str | 描述 |
| settings | str | 生成提示词 |
| image_asset_id | str | 主图素材 ID |
| gen_status | str | none / pending / generating / ready / failed |
| sort_order | int | 排序权重 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

**唯一约束**：`(project_id, name)`

## 8. 用途

场景在分镜层被作为"地点参考"被引用，分镜生成时会同时参考场景图和角色图。
