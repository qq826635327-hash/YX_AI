# 07. 道具

**路由**：
- 列表：`/projects/{projectId}/props`
- 详情：`/projects/{projectId}/props/{propId}`

**对应文件**：
- `pages/PropsPage.tsx`
- `pages/PropDetailPage.tsx`
- 复用组件：[通用组件（EntityCard + EntityEditDialog）](./16-通用组件.md)

## 1. 功能描述

道具模块管理项目中的所有"物品/物件"资产，如武器、工具、装饰品、关键证物等。

## 2. 与场景/角色模块的差异

- 卡片图标不同（Package vs Map vs Users）
- 视觉强调略不同
- **功能完全一致**：列表 / 详情 / 编辑 / 生成 / 删除

## 3. 道具卡片

- 道具主图
- 道具名
- 描述 / 设定
- 操作按钮（生成/编辑/删除）

## 4. 道具编辑字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | text | ✅ | 项目内唯一 |
| 描述 | textarea | ❌ | 多行 |
| 设定 | textarea | ❌ | 用于生成 |

## 5. 道具详情页

同 [角色详情页结构](./05-角色.md#4-角色详情页)，但：
- 顶部图标为包裹（Package）
- 关联分镜：显示使用了此道具的分镜

## 6. 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/projects/{id}/props` | 列表 |
| POST | `/api/projects/{id}/props` | 新建 |
| GET | `/api/projects/{id}/props/{pid}` | 详情 |
| PATCH | `/api/projects/{id}/props/{pid}` | 更新 |
| DELETE | `/api/projects/{id}/props/{pid}` | 删除 |

## 7. 数据模型（Prop）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | UUID |
| project_id | str | 所属项目 |
| name | str | 名称 |
| description | str | 描述 |
| settings | str | 生成提示词 |
| image_asset_id | str | 主图素材 ID |
| gen_status | str | none / pending / generating / ready / failed |
| sort_order | int | 排序权重 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

**唯一约束**：`(project_id, name)`

## 8. 用途

道具在分镜层被作为"物品参考"被引用，特别适合"角色手持某物"、"特写镜头"等场景。
