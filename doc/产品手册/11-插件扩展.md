## 1. 功能描述

管理可插拔的 Provider Handler（实现层）。每个 Handler 是一个 Python 类，实现 `BaseProvider` 接口。

## 6. ProviderHandler 接口

```python
class ProviderHandler(ABC):
    PROVIDER_KIND: str
    SUPPORTED_MODELS: dict

    @abstractmethod
    def translate(self, request: StandardGenerateRequest) -> tuple[str, dict, dict]:
        """标准请求 → 厂商 API 请求（url, payload, headers）"""
        ...

    @abstractmethod
    def parse(self, response_body: dict, status_code: int) -> StandardGenerateResult:
        """厂商响应 → 标准结果"""
        ...

    async def generate_image(self, request: StandardGenerateRequest) -> StandardGenerateResult:
        """基类通用实现：translate → httpx → parse，子类无需覆盖"""
        ...
```
