# 13. 提示词模版

> ⚠️ **[Phase 2 占位 · 数据结构已设计]** 模板变量语法已规划，UI 为占位页面，渲染逻辑待实现。

**路由**：`/settings/prompts`
**对应文件**：`pages/settings/SettingsPromptsPage.tsx`

## 1. 功能描述

管理可复用的提示词模版（如人物一致性提示词、镜头风格提示词等）。

## 2. 页面元素

### 2.1 模版列表
| 列 | 内容 |
|----|------|
| 名称 | 模版唯一名 |
| 用途描述 | 简短说明 |
| 变量占位符 | `{{var_name}}` 列表 |
| 预览 | 渲染样例 |
| 操作 | 编辑/删除/复制 |

### 2.2 新建模版
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | text | ✅ | 唯一 |
| 用途描述 | text | ❌ | - |
| 模板内容 | textarea | ✅ | 含 `{{var_name}}` 占位符 |
| 分类 | select | ❌ | character / scene / shot / style |

## 3. 模板变量

- 用 `{{var_name}}` 标识
- 运行时通过 UI 表单输入
- 支持嵌套引用其他模版

### 3.1 变量语法
```
{{var_name}}                    // 简单替换
{{character.name}}              // 嵌套对象访问
{{#if condition}}...{{/if}}     // 条件块（未来）
```

## 4. 内置模版（Phase 2 计划）

### 4.1 角色一致性模板
```
masterpiece, best quality, {{character.name}},
{{character.description}},
{{character.settings}},
consistent appearance, identity preservation
```

### 4.2 场景一致性模板
```
{{scene.name}}, {{scene.description}},
{{scene.settings}},
consistent lighting, consistent architecture
```

### 4.3 镜头语言模板

| 模板 | 内容 |
|------|------|
| close_up | close-up shot, face focus, emotional detail |
| medium_shot | medium shot, waist up, balanced composition |
| wide_shot | wide shot, full body, environmental context |
| over_shoulder | over-the-shoulder shot, depth of field |

### 4.4 风格化模板
- 写实风：photorealistic, 8k, cinematic
- 动漫风：anime style, vibrant colors, clean lines
- 水墨风：Chinese ink painting, minimalist
- 赛博风：cyberpunk, neon lights, futuristic

## 5. 使用场景

### 5.1 角色详情页
角色编辑时的"settings" 字段可引用角色一致性模板，渲染时自动替换 `{{character.name}}` 等变量。

### 5.2 分镜提示词
分镜的 `first_frame_prompt` 等字段可引用风格化模板 + 角色/场景描述。

### 5.3 实体选择器
在 [剧集结构](./08-剧集结构.md) 选择角色时，自动将角色描述拼入提示词。

## 6. 数据模型（计划）

```python
class PromptTemplate:
    id: str
    name: str           # 唯一
    description: str
    category: str       # character / scene / shot / style
    content: str        # 含 {{var}}
    variables: list     # 自动解析
    builtin: bool       # 内置/用户
    created_at: datetime
    updated_at: datetime
```
