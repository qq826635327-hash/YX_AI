# 18. 素材画廊（AssetGallery）

**对应文件**：`components/AssetGallery.tsx`

## 1. 功能描述

展示某个实体（角色/场景/道具/分镜）关联的所有历史素材（图片 + 视频）。

## 2. 视觉结构

- 网格布局的缩略图列表
- 顶部：标题、过滤/排序、批量操作
- 每个素材项：缩略图、文件名、上传时间、类型徽章

## 3. 功能

### 3.1 缩略图交互
- **单击图片**：弹出 [图片灯箱（ImageLightbox）](./20-图片灯箱.md)大图
- **悬停**：显示操作按钮（设为默认/删除）

### 3.2 选择
- 多选模式
- 复选框显示
- 批量删除

### 3.3 排序
按时间/类型/名称排序

### 3.4 上传
- 拖拽上传
- 点击上传
- 上传进度展示
- 上传完成后自动刷新

### 3.5 设为默认
- 角色/场景/道具详情页可指定某素材为主图
- 点击星标图标切换
- 同步更新实体的 `image_asset_id` 字段

### 3.6 双向同步
- 点击"同步"按钮触发双向同步：
  - **清理**：删除磁盘文件已丢失的 Asset DB 记录
  - **发现**：扫描磁盘上 `角色/`、`场景/`、`道具/`、`分镜/` 目录下的 `images/` 和 `videos/` 子目录，为没有 DB 记录的文件自动创建 Asset 并回填到实体
- 同步完成后 toast 提示：`同步完成：清理 X 条无效记录，发现 Y 个新素材`

## 4. 素材类型

| 类型 | 支持格式 |
|------|----------|
| 图片 | PNG / JPG / WebP / GIF |
| 视频 | MP4 / WebM / MOV / MKV |

## 5. 后端 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/assets?project_id=&target_type=&target_id=` | 列表 |
| GET | `/api/assets/{id}` | 获取单个素材元数据 |
| POST | `/api/assets/projects/{id}/upload?category=&target_type=&target_id=` | 上传新素材 |
| POST | `/api/assets/{id}/upload` | 替换已有素材文件 |
| GET | `/api/assets/{id}/file` | 获取文件 |
| POST | `/api/assets/sync` | 同步清理（清理丢失记录 + 发现磁盘新文件） |
| DELETE | `/api/assets/{id}?delete_file=true` | 删除 |

## 6. 修复历史

### 2026-06-20：删除资产 500 bug 修复
- **问题**：commit 后访问已 expire 的 ORM 属性导致崩溃
- **影响**：所有图片/视频删除都会 500
- **修复**：
  - 改为先缓存所有所需字段再 commit
  - 文件删除独立 try-except
  - Windows 文件被占用不再阻塞数据库操作
  - 路径校验抛 ValueError 也不影响数据库

## 7. 数据模型（Asset）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | UUID |
| project_id | str | 所属项目 |
| asset_type | str | image / video |
| category | str | 主图 / 素材 / 参考图等 |
| file_path | str | 相对项目根的路径 |
| file_name | str | 原始文件名 |
| file_size | int | 字节数 |
| mime_type | str | MIME |
| target_type | str | 关联目标类型（character / scene / prop / shot） |
| target_id | str | 关联目标 ID |
| status | str | pending / ready / failed |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |
