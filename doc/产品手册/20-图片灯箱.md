# 20. 图片灯箱（ImageLightbox）

**对应文件**：`components/ImageLightbox.tsx`

## 1. 功能描述

点击任意图片时弹出全屏大图查看。

## 2. 功能

### 2.1 基础
- 全屏黑色背景
- 图片居中显示
- 关闭按钮（ESC 或点击背景）

### 2.2 缩放控制
- 鼠标滚轮缩放
- 工具栏缩放按钮
- 双击切换 1x/2x
- 拖动平移

### 2.3 文件信息
- 文件名
- 尺寸（像素）
- 文件大小
- 上传时间
- 生成参数

### 2.4 下载
- 下载按钮
- 下载原图

## 3. 视频支持

灯箱内同时支持视频预览：
- 显示播放器控制条
- 视频自动播放（静音）
- 循环播放
- 全屏播放

## 4. 多图切换

当从素材画廊触发时支持：
- 上一张/下一张
- 键盘左右箭头
- 当前张数 / 总数

## 5. 交互

| 操作 | 行为 |
|------|------|
| ESC | 关闭 |
| 点击背景 | 关闭 |
| 滚轮 | 缩放 |
| 双击 | 1x ↔ 2x |
| 拖动 | 平移 |
| ←/→ | 上一张/下一张 |
| D 键 | 下载 |

## 6. 视觉

- 黑色 80% 透明背景
- 图片最大占视口 90%
- 控制条顶部悬浮
- 关闭按钮右上角
