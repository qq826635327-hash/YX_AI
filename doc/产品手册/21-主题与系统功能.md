# 21. 主题与系统功能

**对应文件**：`components/layout/`，`stores/ui.ts`

## 1. 主题切换

### 1.1 入口
Sidebar 底部主题按钮

### 1.2 三态循环
- 浅色模式（light）
- 深色模式（dark）
- 跟随系统（system）

### 1.3 持久化
- localStorage 存储用户选择
- 刷新后保留
- 跟随系统时监听 prefers-color-scheme

### 1.4 实现
- 基于 CSS 变量（`index.css` 中定义 light/dark token）
- Tailwind 工具类 + CSS 变量
- Radix UI 原语原生支持主题

## 2. 全局功能

### 2.1 自动保存
- 除剧本等大型编辑外，其他修改即时保存
- 提示词输入框带 debounce 自动保存
- PATCH 失败时显示 Toast 错误

### 2.2 WebSocket 实时同步
- 任务进度自动推送
- 剧本解析进度自动推送
- 新任务创建立即收到

### 2.3 轮询降级
- WS 断开时自动启用 5 秒轮询
- 通过检测 `WsClient.disconnected` 状态触发
- 重新连接后停止轮询

### 2.4 Toast 通知
- 位置：右上角
- 类型：success / error / info / warning
- 自动消失：默认 3 秒
- 手动关闭
- 同类型不堆叠

### 2.5 ErrorBoundary
- 包裹整个应用
- 异常时显示降级页
- 提供"刷新页面"和"复制错误"按钮
- 不暴露敏感堆栈
