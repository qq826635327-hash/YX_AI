## B.3 Phase 详细规划

### Phase 1（当前已完成）
- 项目骨架：FastAPI + React + Vite + Tailwind
- 完整 CRUD：项目/角色/场景/道具/剧集/分镜/任务/配置
- WebSocket 实时推送
- 双轨状态同步（WS + 轮询）
- API Key Fernet 加密
- 主题切换

### Phase 2（剧本解析 LLM 化）
- 接入 LLM（OpenAI / Claude / 本地 LLM）
- 结构化输出（Function Calling / JSON Schema）
- 实体自动抽取：角色/场景/道具/剧集/分镜
- 提示词模版完整实现
- 解析失败的回退机制

### Phase 3（真实生成引擎）
- ComfyUI provider 实现
- API provider 实现（Agnes / SenseNova）
- 工作流实际提交 + WebSocket 进度
- 真实生成图片/视频
- 模型配置自动发现
- 重试机制完善

### Phase 4（素材与多模态）
- 资产版本管理（同一实体的多版主图）
- 视频时间线编辑器
- 跨项目素材复用
- 数据导出（ZIP 包）
- 插件扩展机制

### Phase 5（配置中心完善）
- ComfyUI 工作流可视化编辑器
- 节点拖拽
- 参数面板
- 实时预览
