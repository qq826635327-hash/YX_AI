## AI 使用指南

本手册面向 AI 编程助手，每个模块文档末尾都有 **"AI 开发检查清单"** 章节。

**工作流程**：
1. 先读 `doc/记忆文件/Ai执行必读.md` 了解工作流和强制规则
2. 修改某模块前，读该模块的产品手册 + 对应开发手册章节
3. 修改完成后，逐项执行检查清单
4. 全部通过后，更新 CHANGELOG.md

## 维护规则

- **改前端页面**：找到对应的 `XX-模块名.md`（如改角色页 → `05-角色.md`）
- **改后端 API**：更新 `A-完整路由表.md`
- **改架构 / 重构**：更新 [开发手册 CHANGELOG](../开发手册/CHANGELOG.md)
- **新发现 Bug**：更新 [开发手册 08-常见Bug与注意事项](../开发手册/08-常见Bug与注意事项.md)

*本手册由各模块独立 md 文件组成，便于按需查阅和更新。*
