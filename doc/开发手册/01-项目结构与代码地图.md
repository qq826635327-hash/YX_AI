# 01 · 项目结构与代码地图

> 找代码用这章。新人/AI 改任何东西前先查这里定位。

## 1. 后端目录（`backend/app/`）

```
app/
├── main.py                 # FastAPI 入口、lifespan、CORS、BasicAuth、静态托管
├── db.py                   # 引擎、会话、WAL 模式
│
├── core/                   # 核心工具（与业务无关）
│   ├── config.py           # 配置加载（YAML + 环境变量）、Fernet 加密
│   ├── serialization.py    # 序列化工具（datetime → ISO 字符串）
│   └── trace.py            # ★ trace_id 全链路追踪（contextvars）
│
├── models/                 # SQLModel 表模型
│   ├── base.py             # IDMixin, TimestampMixin, gen_uuid, utcnow
│   ├── project.py          # Project
│   ├── script.py           # ScriptDocument（剧本）
│   ├── character.py        # Character（角色）
│   ├── scene.py            # Scene
│   ├── prop.py             # Prop
│   ├── episode.py          # Episode（剧集）
│   ├── shot.py             # Shot（分镜）
│   ├── shot_reference.py   # ShotCharacter, ShotScene, ShotProp（分镜关联表）
│   ├── asset.py            # Asset（图片/视频素材）
│   ├── task.py             # GenerationTask（生成任务）
│   ├── task_log.py         # TaskLog（任务日志，DEBUG/INFO/WARNING/ERROR）
│   ├── provider.py         # ApiProvider（API 供应商）
│   └── workflow.py         # WorkflowMapping（ComfyUI 工作流映射）
│
├── schemas/                # Pydantic 请求/响应模型
│   ├── common.py           # 通用
│   ├── project.py
│   ├── script.py
│   ├── business.py         # 角色/场景/道具/剧集/分镜 通用 schema
│   ├── generation.py       # 生成任务
│   └── provider_types.py   # ★ StandardGenerateRequest / StandardGenerateResult / ModelCapabilities
│
├── services/               # 业务逻辑层（无 HTTP 细节）
│   ├── business_service.py # ★ 通用 CRUD（list_by_project / create_entity / update_entity / delete_entity）
│   ├── project_service.py
│   ├── asset_service.py    # 资产创建/删除/路径解析（★ 删除逻辑含文件清理）
│   ├── generation_service.py
│   ├── config_service.py   # Provider 配置 + 密钥加解密
│   ├── script_service.py   # 剧本解析业务
│   ├── shot_reference_service.py
│   └── task_log_service.py # ★ 结构化任务日志（log_api_call / log_ref_collect / log_download + WS 推送）
│
├── api/                    # FastAPI 路由层（只做参数解析 + 调 service）
│   ├── __init__.py         # api_router 聚合
│   ├── projects.py
│   ├── script.py
│   ├── characters.py       # ★ 调用 business_service 通用 CRUD
│   ├── scenes.py           # ★ 同上
│   ├── props.py            # ★ 同上
│   ├── episodes.py         # ★ 同上
│   ├── shots.py            # ★ 同上
│   ├── shot_references.py
│   ├── assets.py
│   ├── generate.py
│   ├── tasks.py
│   ├── providers.py
│   ├── workflows.py
│   ├── settings.py
│   └── logs.py             # ★ 日志查询 API（历史 / 元信息 / 清空）
│
├── tasks/                  # 后台任务
│   └── execute_task.py     # 任务执行器（Huey 调度）
│
├── pipelines/              # 复杂数据处理
│   └── script_parser.py    # 剧本解析（LLM 调用）
│
├── providers/              # AI Provider 抽象（translate/parse 模式）
│   ├── base.py             # ★ ProviderHandler 抽象类（translate/parse + 基类 generate_image）
│   ├── registry.py         # Handler 注册表（@register 装饰器）
│   ├── agnes_handler.py    # Agnes / OpenAI 兼容实现
│   └── __init__.py
│
├── clients/                # 第三方 API 客户端
│   └── agnes_client.py     # Agnes HTTP 客户端（图片下载）
│
└── ws/                     # WebSocket
    ├── routes.py           # /ws/tasks /ws/script /ws/logs 路由
    └── log_handler.py      # ★ WsLogHandler：把 ERROR/WARNING 广播到 /ws/logs
```

## 3. "我要加新功能去哪改" 速查

| 我要加 | 改哪里 |
| --- | --- |
| 新建一张数据库表 | `models/<name>.py` → `models/__init__.py` 导出 |
| 新建一种业务实体（用户管理、标签、笔记……） | `models/` + `services/business_service.py`（如果只是 4-5 字段的 CRUD，直接复用） + `api/<name>.py` |
| 新建一个 AI Provider | `providers/<name>_handler.py`（继承 `ProviderHandler`）+ `providers/__init__.py` 添加 import 注册 |
| 新建一个生成任务类型（音频/字幕……） | `models/task.py`（扩展 `TARGET_TYPES`）+ `api/generate.py` + `tasks/execute_task.py` |
| 新建一个 WebSocket 频道 | `ws/routes.py`（参考 `/ws/logs` 实现） |
| 新建一个前端页面 | `pages/<Name>Page.tsx` + `App.tsx` 加路由 + `Sidebar.tsx` 加菜单 |
| 新建一个可复用组件 | `components/<Name>.tsx` |
| 加一个全局配置项 | `core/config.py`（`AppConfig`） + `config.yaml` |
| 加一个 API 路径 | `api/<name>.py` + `api/__init__.py` 包含 + 同步 `doc/产品手册/A-完整路由表.md` |
| 修改日志输出 | 看 `07-日志监控开发指南.md` |

## 4. 数据库 ER 简图

```
Project (1) ──< (N) ScriptDocument
Project (1) ──< (N) Character
Project (1) ──< (N) Scene
Project (1) ──< (N) Prop
Project (1) ──< (N) Episode (1) ──< (N) Shot
Project (1) ──< (N) Asset
Project (1) ──< (N) GenerationTask
Shot (M) ──< (M) Character  (通过 ShotCharacter)
Shot (M) ──< (M) Scene      (通过 ShotScene)
Shot (M) ──< (M) Prop       (通过 ShotProp)
GenerationTask (1) ──< (N) TaskLog
GenerationTask (1) ──< (1) Asset (output_asset_id)
ApiProvider (独立表；与 Project 通过 GenerationTask 关联)
WorkflowMapping (独立表)
```
