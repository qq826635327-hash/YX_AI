### 3.1 项目管理 API

#### 3.1.1 POST /api/projects - 创建项目

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-PROJ-001 | 创建项目-正常路径 | 无 | POST {"name": "测试项目", "description": "描述"} | 200, 返回项目对象, status=active | P0 |
| TC-API-PROJ-002 | 创建项目-缺少名称 | 无 | POST {"description": "描述"} | 422, name 为必填字段 | P0 |
| TC-API-PROJ-003 | 创建项目-名称为空字符串 | 无 | POST {"name": ""} | 422, 名称不能为空 | P1 |
| TC-API-PROJ-004 | 创建项目-名称超长 | 无 | POST {"name": "x"*500} | 422, 名称超出长度限制 | P1 |
| TC-API-PROJ-005 | 创建项目-含特殊字符 | 无 | POST {"name": "<script>alert(1)</script>"} | 200, 名称正确存储（无XSS转义由前端处理） | P2 |
| TC-API-PROJ-006 | 创建项目-仅必填字段 | 无 | POST {"name": "最小项目"} | 200, description 为 null/空 | P1 |
| TC-API-PROJ-007 | 创建项目-重复名称 | 已存在同名项目 | POST {"name": "已存在项目"} | 200 或 409（取决于业务规则） | P2 |

```python
# tests/api/test_projects.py
class TestCreateProject:
    """POST /api/projects 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_proj_001_create_success(self, client: AsyncClient):
        """TC-API-PROJ-001: 创建项目-正常路径"""
        response = await client.post("/api/projects", json={
            "name": "测试项目",
            "description": "自动化测试项目"
        })
        assert response.status_code == 200
        data = response.json()["data"]
        assert data["name"] == "测试项目"
        assert data["description"] == "自动化测试项目"
        assert data["status"] == "active"
        assert "id" in data
        assert "created_at" in data

    @pytest.mark.asyncio
    async def test_tc_api_proj_002_missing_name(self, client: AsyncClient):
        """TC-API-PROJ-002: 缺少名称字段"""
        response = await client.post("/api/projects", json={
            "description": "没有名称"
        })
        assert response.status_code == 422

    @pytest.mark.asyncio
    async def test_tc_api_proj_003_empty_name(self, client: AsyncClient):
        """TC-API-PROJ-003: 名称为空字符串"""
        response = await client.post("/api/projects", json={
            "name": "",
            "description": "空名称"
        })
        assert response.status_code == 422

    @pytest.mark.asyncio
    async def test_tc_api_proj_004_name_too_long(self, client: AsyncClient):
        """TC-API-PROJ-004: 名称超长"""
        response = await client.post("/api/projects", json={
            "name": "x" * 500
        })
        assert response.status_code == 422

    @pytest.mark.asyncio
    async def test_tc_api_proj_006_minimal_fields(self, client: AsyncClient):
        """TC-API-PROJ-006: 仅必填字段"""
        response = await client.post("/api/projects", json={
            "name": "最小项目"
        })
        assert response.status_code == 200
        data = response.json()["data"]
        assert data["name"] == "最小项目"
```

#### 3.1.2 GET /api/projects - 获取项目列表

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-PROJ-010 | 获取项目列表-有数据 | 已创建3个项目 | GET /api/projects | 200, 返回3个项目 | P0 |
| TC-API-PROJ-011 | 获取项目列表-无数据 | 无任何项目 | GET /api/projects | 200, 返回空列表 | P0 |
| TC-API-PROJ-012 | 获取项目列表-分页 | 已创建15个项目 | GET /api/projects?page=2&size=5 | 200, 返回第2页5条数据 | P1 |
| TC-API-PROJ-013 | 获取项目列表-排序 | 已创建多个项目 | GET /api/projects?sort=created_at&order=desc | 200, 按创建时间倒序 | P2 |
| TC-API-PROJ-014 | 获取项目列表-筛选状态 | 有active和archived项目 | GET /api/projects?status=active | 200, 仅返回active项目 | P2 |

```python
class TestListProjects:
    """GET /api/projects 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_proj_010_list_with_data(
        self, client: AsyncClient, project_factory
    ):
        """TC-API-PROJ-010: 获取项目列表-有数据"""
        # 创建3个测试项目
        project_factory.create_batch(3)
        response = await client.get("/api/projects")
        assert response.status_code == 200
        data = response.json()["data"]
        assert len(data) >= 3

    @pytest.mark.asyncio
    async def test_tc_api_proj_011_list_empty(self, client: AsyncClient):
        """TC-API-PROJ-011: 获取项目列表-无数据"""
        response = await client.get("/api/projects")
        assert response.status_code == 200
        data = response.json()["data"]
        assert isinstance(data, list)
        assert len(data) == 0
```

#### 3.1.4 PATCH /api/projects/{id} - 更新项目

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-PROJ-030 | 更新项目-修改名称 | 项目已创建 | PATCH {"name": "新名称"} | 200, 名称已更新 | P0 |
| TC-API-PROJ-031 | 更新项目-修改状态为archived | 项目status=active | PATCH {"status": "archived"} | 200, 状态已更新 | P0 |
| TC-API-PROJ-032 | 更新项目-不存在 | 无 | PATCH /api/projects/99999 | 404 | P0 |
| TC-API-PROJ-033 | 更新项目-无效状态值 | 项目已创建 | PATCH {"status": "invalid"} | 422, 枚举值校验失败 | P1 |
| TC-API-PROJ-034 | 更新项目-部分更新 | 项目已创建 | PATCH {"description": "新描述"} | 200, 仅description更新 | P1 |
| TC-API-PROJ-035 | 更新项目-空body | 项目已创建 | PATCH {} | 200 或 422 | P2 |

```python
class TestUpdateProject:
    """PATCH /api/projects/{id} 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_proj_030_update_name(
        self, client: AsyncClient, project
    ):
        """TC-API-PROJ-030: 更新项目名称"""
        response = await client.patch(
            f"/api/projects/{project.id}",
            json={"name": "更新后的名称"}
        )
        assert response.status_code == 200
        assert response.json()["data"]["name"] == "更新后的名称"

    @pytest.mark.asyncio
    async def test_tc_api_proj_031_archive_project(
        self, client: AsyncClient, project
    ):
        """TC-API-PROJ-031: 归档项目"""
        response = await client.patch(
            f"/api/projects/{project.id}",
            json={"status": "archived"}
        )
        assert response.status_code == 200
        assert response.json()["data"]["status"] == "archived"

    @pytest.mark.asyncio
    async def test_tc_api_proj_033_invalid_status(
        self, client: AsyncClient, project
    ):
        """TC-API-PROJ-033: 无效状态值"""
        response = await client.patch(
            f"/api/projects/{project.id}",
            json={"status": "deleted"}
        )
        assert response.status_code == 422
```

#### 3.1.5 DELETE /api/projects/{id} - 删除项目

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-PROJ-040 | 删除项目-正常 | 项目已创建，无关联数据 | DELETE /api/projects/{id} | 200/204, 项目已删除 | P0 |
| TC-API-PROJ-041 | 删除项目-不存在 | 无 | DELETE /api/projects/99999 | 404 | P0 |
| TC-API-PROJ-042 | 删除项目-含关联数据 | 项目含角色/场景/剧集 | DELETE /api/projects/{id} | 级联删除或 409 冲突 | P1 |
| TC-API-PROJ-043 | 删除后验证 | 项目已删除 | GET /api/projects/{id} | 404 | P0 |

```python
class TestDeleteProject:
    """DELETE /api/projects/{id} 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_proj_040_delete_success(
        self, client: AsyncClient, project
    ):
        """TC-API-PROJ-040: 正常删除项目"""
        response = await client.delete(f"/api/projects/{project.id}")
        assert response.status_code in [200, 204]
        # 验证删除
        verify = await client.get(f"/api/projects/{project.id}")
        assert verify.status_code == 404

    @pytest.mark.asyncio
    async def test_tc_api_proj_041_delete_not_found(
        self, client: AsyncClient
    ):
        """TC-API-PROJ-041: 删除不存在的项目"""
        response = await client.delete("/api/projects/99999")
        assert response.status_code == 404
```
