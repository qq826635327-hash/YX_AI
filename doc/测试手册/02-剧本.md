### 3.2 剧本 API

#### 3.2.1 GET /api/projects/{id}/script - 获取剧本

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCRIPT-001 | 获取剧本-有内容 | 已保存剧本内容 | GET /api/projects/{id}/script | 200, 返回剧本数据 | P0 |
| TC-API-SCRIPT-002 | 获取剧本-无内容 | 项目已创建，无剧本 | GET /api/projects/{id}/script | 200, 返回空/null | P0 |
| TC-API-SCRIPT-003 | 获取剧本-项目不存在 | 无 | GET /api/projects/99999/script | 404 | P0 |

#### 3.2.2 PUT /api/projects/{id}/script - 更新剧本

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCRIPT-010 | 更新剧本-正常 | 项目已创建 | PUT {"content": "剧本内容..."} | 200, 剧本已保存 | P0 |
| TC-API-SCRIPT-011 | 更新剧本-超长内容 | 项目已创建 | PUT 10万字内容 | 200, 正确存储 | P1 |
| TC-API-SCRIPT-012 | 更新剧本-特殊字符 | 项目已创建 | PUT 含 \n\t\" 内容 | 200, 正确转义存储 | P1 |
| TC-API-SCRIPT-013 | 更新剧本-项目不存在 | 无 | PUT /api/projects/99999/script | 404 | P0 |

#### 3.2.3 POST /api/projects/{id}/script/parse - 解析剧本

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCRIPT-020 | 解析剧本-正常格式 | 已有标准格式剧本 | POST /parse | 200, 返回解析结果 | P0 |
| TC-API-SCRIPT-021 | 解析剧本-空内容 | 剧本内容为空 | POST /parse | 400/422, 提示无内容 | P0 |
| TC-API-SCRIPT-022 | 解析剧本-非标准格式 | 剧本格式不规范 | POST /parse | 200, 尽力解析 + 警告 | P1 |
| TC-API-SCRIPT-023 | 解析剧本-触发WebSocket | 已连接WS | POST /parse | 解析进度通过 /ws/script 推送 | P1 |

```python
class TestScriptAPI:
    """剧本 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_script_001_get_script(
        self, client: AsyncClient, project_with_script
    ):
        """TC-API-SCRIPT-001: 获取已有剧本"""
        response = await client.get(
            f"/api/projects/{project_with_script.id}/script"
        )
        assert response.status_code == 200
        assert response.json()["data"]["content"] is not None

    @pytest.mark.asyncio
    async def test_tc_api_script_010_update_script(
        self, client: AsyncClient, project
    ):
        """TC-API-SCRIPT-010: 更新剧本"""
        script_content = "第一幕\n场景：城市街道\n角色A：你好..."
        response = await client.put(
            f"/api/projects/{project.id}/script",
            json={"content": script_content}
        )
        assert response.status_code == 200

    @pytest.mark.asyncio
    async def test_tc_api_script_020_parse_script(
        self, client: AsyncClient, project_with_script
    ):
        """TC-API-SCRIPT-020: 解析剧本"""
        response = await client.post(
            f"/api/projects/{project_with_script.id}/script/parse"
        )
        assert response.status_code == 200
        data = response.json()["data"]
        assert "episodes" in data or "scenes" in data
```
