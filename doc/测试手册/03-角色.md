### 3.3 角色管理 API

#### 3.3.1 POST /api/projects/{id}/characters - 创建角色

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-CHAR-001 | 创建主角 | 项目已创建 | POST {"name": "主角A", "char_type": "protagonist"} | 200, 角色创建成功 | P0 |
| TC-API-CHAR-002 | 创建配角 | 项目已创建 | POST {"name": "配角B", "char_type": "supporting"} | 200 | P0 |
| TC-API-CHAR-003 | 创建群演 | 项目已创建 | POST {"name": "路人C", "char_type": "extra"} | 200 | P1 |
| TC-API-CHAR-004 | 缺少名称 | 项目已创建 | POST {"char_type": "protagonist"} | 422 | P0 |
| TC-API-CHAR-005 | 无效类型 | 项目已创建 | POST {"name": "X", "char_type": "invalid"} | 422 | P1 |
| TC-API-CHAR-006 | 项目不存在 | 无 | POST /api/projects/99999/characters | 404 | P0 |
| TC-API-CHAR-007 | 含描述信息 | 项目已创建 | POST 含 description 字段 | 200, 描述正确存储 | P1 |
| TC-API-CHAR-008 | 含设定JSON | 项目已创建 | POST 含 settings 字段 | 200, settings 正确存储 | P2 |

```python
class TestCharacterAPI:
    """角色管理 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_char_001_create_protagonist(
        self, client: AsyncClient, project
    ):
        """TC-API-CHAR-001: 创建主角"""
        response = await client.post(
            f"/api/projects/{project.id}/characters",
            json={
                "name": "林小雨",
                "char_type": "protagonist",
                "description": "25岁女程序员，性格内向但才华横溢"
            }
        )
        assert response.status_code == 200
        data = response.json()["data"]
        assert data["name"] == "林小雨"
        assert data["char_type"] == "protagonist"
        assert data["gen_status"] == "none"

    @pytest.mark.asyncio
    async def test_tc_api_char_004_missing_name(
        self, client: AsyncClient, project
    ):
        """TC-API-CHAR-004: 缺少名称"""
        response = await client.post(
            f"/api/projects/{project.id}/characters",
            json={"char_type": "protagonist"}
        )
        assert response.status_code == 422

    @pytest.mark.asyncio
    async def test_tc_api_char_005_invalid_type(
        self, client: AsyncClient, project
    ):
        """TC-API-CHAR-005: 无效角色类型"""
        response = await client.post(
            f"/api/projects/{project.id}/characters",
            json={"name": "测试", "char_type": "villain"}
        )
        assert response.status_code == 422
```

#### 3.3.4 PATCH /api/projects/{id}/characters/{cid} - 更新角色

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-CHAR-030 | 更新角色名称 | 角色已创建 | PATCH {"name": "新名称"} | 200, 名称已更新 | P0 |
| TC-API-CHAR-031 | 更新角色类型 | 角色char_type=supporting | PATCH {"char_type": "protagonist"} | 200 | P1 |
| TC-API-CHAR-032 | 更新描述 | 角色已创建 | PATCH {"description": "新描述"} | 200 | P1 |
| TC-API-CHAR-033 | 更新排序 | 角色已创建 | PATCH {"sort_order": 5} | 200 | P2 |
| TC-API-CHAR-034 | 角色不存在 | 无 | PATCH /characters/99999 | 404 | P0 |
| TC-API-CHAR-035 | 并发更新 | 角色已创建 | 同时发送2个PATCH请求 | 最后写入者胜出，无数据损坏 | P2 |

```python
class TestUpdateCharacter:
    """PATCH /api/projects/{id}/characters/{cid} 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_char_030_update_name(
        self, client: AsyncClient, project, character
    ):
        """TC-API-CHAR-030: 更新角色名称"""
        response = await client.patch(
            f"/api/projects/{project.id}/characters/{character.id}",
            json={"name": "新角色名"}
        )
        assert response.status_code == 200
        assert response.json()["data"]["name"] == "新角色名"

    @pytest.mark.asyncio
    async def test_tc_api_char_035_concurrent_update(
        self, client: AsyncClient, project, character
    ):
        """TC-API-CHAR-035: 并发更新测试"""
        import asyncio
        tasks = [
            client.patch(
                f"/api/projects/{project.id}/characters/{character.id}",
                json={"description": f"描述{i}"}
            )
            for i in range(5)
        ]
        results = await asyncio.gather(*tasks)
        # 所有请求应成功，无数据损坏
        assert all(r.status_code == 200 for r in results)
        # 验证最终状态一致
        verify = await client.get(
            f"/api/projects/{project.id}/characters/{character.id}"
        )
        assert verify.status_code == 200
```
