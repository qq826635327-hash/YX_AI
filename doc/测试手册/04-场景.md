### 3.4 场景管理 API

#### 3.4.1 POST /api/projects/{id}/scenes - 创建场景

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCENE-001 | 创建场景-正常 | 项目已创建 | POST {"name": "城市街道"} | 200 | P0 |
| TC-API-SCENE-002 | 缺少名称 | 项目已创建 | POST {"description": "..."} | 422 | P0 |
| TC-API-SCENE-003 | 项目不存在 | 无 | POST /api/projects/99999/scenes | 404 | P0 |
| TC-API-SCENE-004 | 含描述和设定 | 项目已创建 | POST 含 description, settings | 200 | P1 |
| TC-API-SCENE-005 | 重复名称 | 已有同名场景 | POST {"name": "已存在场景"} | 200 或 409 | P2 |

#### 3.4.2 GET /api/projects/{id}/scenes - 获取场景列表

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCENE-010 | 获取场景列表 | 项目含多个场景 | GET /scenes | 200, 返回所有场景 | P0 |
| TC-API-SCENE-011 | 空列表 | 项目无场景 | GET /scenes | 200, [] | P0 |
| TC-API-SCENE-012 | 项目不存在 | 无 | GET /api/projects/99999/scenes | 404 | P0 |

#### 3.4.3 GET/PATCH/DELETE /api/projects/{id}/scenes/{sid}

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SCENE-020 | 获取场景详情 | 场景已创建 | GET /scenes/{sid} | 200 | P0 |
| TC-API-SCENE-021 | 场景不存在 | 无 | GET /scenes/99999 | 404 | P0 |
| TC-API-SCENE-030 | 更新场景 | 场景已创建 | PATCH {"name": "新名称"} | 200 | P0 |
| TC-API-SCENE-031 | 更新不存在场景 | 无 | PATCH /scenes/99999 | 404 | P0 |
| TC-API-SCENE-040 | 删除场景 | 场景已创建 | DELETE /scenes/{sid} | 200/204 | P0 |
| TC-API-SCENE-041 | 删除不存在场景 | 无 | DELETE /scenes/99999 | 404 | P0 |
| TC-API-SCENE-042 | 删除被引用场景 | 场景已关联分镜 | DELETE /scenes/{sid} | 级联处理或 409 | P1 |

```python
class TestSceneAPI:
    """场景管理 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_scene_001_create(
        self, client: AsyncClient, project
    ):
        """TC-API-SCENE-001: 创建场景"""
        response = await client.post(
            f"/api/projects/{project.id}/scenes",
            json={
                "name": "雨夜小巷",
                "description": "昏暗的路灯下，雨水打在青石板上"
            }
        )
        assert response.status_code == 200
        data = response.json()["data"]
        assert data["name"] == "雨夜小巷"
        assert data["gen_status"] == "none"

    @pytest.mark.asyncio
    async def test_tc_api_scene_030_update(
        self, client: AsyncClient, project, scene
    ):
        """TC-API-SCENE-030: 更新场景"""
        response = await client.patch(
            f"/api/projects/{project.id}/scenes/{scene.id}",
            json={"name": "更新后的场景名"}
        )
        assert response.status_code == 200
        assert response.json()["data"]["name"] == "更新后的场景名"
```
