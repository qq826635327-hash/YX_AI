### 3.6 剧集管理 API

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-EP-001 | 创建剧集 | 项目已创建 | POST {"episode_no": 1, "title": "第一集"} | 200 | P0 |
| TC-API-EP-002 | 缺少集数 | 项目已创建 | POST {"title": "第一集"} | 422 | P0 |
| TC-API-EP-003 | 缺少标题 | 项目已创建 | POST {"episode_no": 1} | 422 | P0 |
| TC-API-EP-004 | 重复集数 | 已有第1集 | POST {"episode_no": 1, "title": "重复"} | 409 或 200 | P1 |
| TC-API-EP-005 | 项目不存在 | 无 | POST /api/projects/99999/episodes | 404 | P0 |
| TC-API-EP-010 | 获取剧集列表 | 项目含多集 | GET /episodes | 200, 按 episode_no 排序 | P0 |
| TC-API-EP-011 | 空列表 | 项目无剧集 | GET /episodes | 200, [] | P0 |
| TC-API-EP-020 | 获取剧集详情 | 剧集已创建 | GET /episodes/{eid} | 200 | P0 |
| TC-API-EP-021 | 剧集不存在 | 无 | GET /episodes/99999 | 404 | P0 |
| TC-API-EP-030 | 更新剧集 | 剧集已创建 | PATCH {"title": "新标题"} | 200 | P0 |
| TC-API-EP-031 | 更新剧集-不存在 | 无 | PATCH /episodes/99999 | 404 | P0 |
| TC-API-EP-040 | 删除剧集 | 剧集已创建 | DELETE /episodes/{eid} | 200/204 | P0 |
| TC-API-EP-041 | 删除剧集-不存在 | 无 | DELETE /episodes/99999 | 404 | P0 |
| TC-API-EP-042 | 删除含分镜的剧集 | 剧集含分镜 | DELETE /episodes/{eid} | 级联删除分镜或 409 | P1 |

```python
class TestEpisodeAPI:
    """剧集管理 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_ep_001_create(
        self, client: AsyncClient, project
    ):
        """TC-API-EP-001: 创建剧集"""
        response = await client.post(
            f"/api/projects/{project.id}/episodes",
            json={
                "episode_no": 1,
                "title": "命运的邂逅",
                "summary": "主角在雨中遇到了改变她一生的人"
            }
        )
        assert response.status_code == 200
        data = response.json()
        assert data["episode_no"] == 1
        assert data["title"] == "命运的邂逅"

    @pytest.mark.asyncio
    async def test_tc_api_ep_010_list_ordered(
        self, client: AsyncClient, project
    ):
        """TC-API-EP-010: 剧集列表按集数排序"""
        # 创建多集（乱序）
        for no in [3, 1, 2]:
            await client.post(
                f"/api/projects/{project.id}/episodes",
                json={"episode_no": no, "title": f"第{no}集"}
            )
        response = await client.get(
            f"/api/projects/{project.id}/episodes"
        )
        assert response.status_code == 200
        data = response.json()
        episode_nos = [ep["episode_no"] for ep in data]
        assert episode_nos == sorted(episode_nos)
```
