#### 3.7.2 分镜关联 API

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-SHOTREF-001 | 获取分镜关联 | 分镜已关联角色/场景/道具 | GET /api/shots/{sid}/references | 200, 含所有关联 | P0 |
| TC-API-SHOTREF-002 | 无关联 | 分镜未关联 | GET /api/shots/{sid}/references | 200, 空关联 | P0 |
| TC-API-SHOTREF-010 | 关联角色 | 分镜+角色已创建 | POST /api/shots/{sid}/characters | 200 | P0 |
| TC-API-SHOTREF-011 | 关联不存在的角色 | 分镜已创建 | POST 不存在的角色ID | 404 | P1 |
| TC-API-SHOTREF-012 | 重复关联 | 已关联某角色 | POST 相同角色ID | 409 或幂等 | P2 |
| TC-API-SHOTREF-020 | 取消关联角色 | 已关联角色 | DELETE /api/shots/{sid}/characters | 200/204 | P0 |
| TC-API-SHOTREF-030 | 关联场景 | 分镜+场景已创建 | POST /api/shots/{sid}/scenes | 200 | P0 |
| TC-API-SHOTREF-031 | 取消关联场景 | 已关联场景 | DELETE /api/shots/{sid}/scenes | 200/204 | P0 |
| TC-API-SHOTREF-040 | 关联道具 | 分镜+道具已创建 | POST /api/shots/{sid}/props | 200 | P0 |
| TC-API-SHOTREF-041 | 取消关联道具 | 已关联道具 | DELETE /api/shots/{sid}/props | 200/204 | P0 |

```python
class TestShotAPI:
    """分镜管理 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_shot_001_create(
        self, client: AsyncClient, episode
    ):
        """TC-API-SHOT-001: 创建分镜"""
        response = await client.post(
            f"/api/episodes/{episode.id}/shots",
            json={
                "shot_no": 1,
                "summary": "主角站在窗前，望着外面的雨",
                "first_frame_prompt": "A young woman standing by window, rain outside"
            }
        )
        assert response.status_code == 200
        data = response.json()
        assert data["shot_no"] == 1
        assert data["first_frame_status"] is None or data["first_frame_status"] == "none"

    @pytest.mark.asyncio
    async def test_tc_api_shotref_010_link_character(
        self, client: AsyncClient, shot, character
    ):
        """TC-API-SHOTREF-010: 分镜关联角色"""
        response = await client.post(
            f"/api/shots/{shot.id}/characters",
            json={"character_id": character.id}
        )
        assert response.status_code == 200
        # 验证关联
        refs = await client.get(f"/api/shots/{shot.id}/references")
        assert character.id in [
            c["id"] for c in refs.json().get("characters", [])
        ]
```
