### 3.8 生成任务 API

#### 3.8.1 POST /api/generate - 单任务生成

| 编号             | 名称             | 前置条件                   | 步骤                              | 预期结果                   | 优先级 |
| -------------- | -------------- | ---------------------- | ------------------------------- | ---------------------- | --- |
| TC-API-GEN-001 | 提交生成任务-正常      | 已有可用的 Provider + 分镜    | POST 有效 payload                 | 200, 返回 task_id        | P0  |
| TC-API-GEN-002 | 缺少 target_type | Provider 已配置           | POST 缺少 target_type             | 422                    | P0  |
| TC-API-GEN-003 | 无效 target_type | Provider 已配置           | POST {"target_type": "invalid"} | 422                    | P1  |
| TC-API-GEN-004 | 无效 target_id   | Provider 已配置           | POST {"target_id": 99999}       | 404                    | P0  |
| TC-API-GEN-005 | Provider 未配置   | 无 Provider             | POST 有效 payload                 | 400/503, 提示无可用Provider | P0  |
| TC-API-GEN-006 | Provider 不可用   | Provider enabled=false | POST 有效 payload                 | 400/503                | P1  |
| TC-API-GEN-007 | 图片生成-首帧        | 分镜有 first_frame_prompt | POST target_type=first_frame    | 200, 任务创建成功            | P0  |
| TC-API-GEN-008 | 视频生成           | 分镜有 video_prompt       | POST target_type=video          | 200, 任务创建成功            | P0  |
| TC-API-GEN-009 | 缺少 prompt      | 分镜无 prompt             | POST                            | 400, 提示缺少提示词           | P1  |

```python
class TestGenerationAPI:
    """生成任务 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_gen_001_submit_task(
        self, client: AsyncClient, project, shot_with_prompt, mock_provider
    ):
        """TC-API-GEN-001: 提交生成任务"""
        response = await client.post("/api/generate", json={
            "target_type": "first_frame",
            "target_id": shot_with_prompt.id,
            "provider_id": mock_provider.id
        })
        assert response.status_code == 200
        data = response.json()
        assert "id" in data
        assert data["status"] in ["pending", "queued"]

    @pytest.mark.asyncio
    async def test_tc_api_gen_005_no_provider(
        self, client: AsyncClient, shot_with_prompt
    ):
        """TC-API-GEN-005: 无可用Provider"""
        response = await client.post("/api/generate", json={
            "target_type": "first_frame",
            "target_id": shot_with_prompt.id
        })
        assert response.status_code in [400, 503]
```

#### 3.8.2 POST /api/generate/batch - 批量生成

#### 3.8.3 POST /api/generate/{tid}/retry - 重试任务
