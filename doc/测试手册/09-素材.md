### 3.10 素材管理 API

| 编号 | 名称 | 前置条件 | 步骤 | 预期结果 | 优先级 |
|------|------|----------|------|----------|--------|
| TC-API-ASSET-001 | 获取素材列表 | 有素材 | GET /api/assets | 200 | P0 |
| TC-API-ASSET-002 | 空列表 | 无素材 | GET /api/assets | 200, [] | P0 |
| TC-API-ASSET-003 | 按类型筛选 | 有图片和视频 | GET /api/assets?asset_type=image | 200, 仅图片 | P1 |
| TC-API-ASSET-004 | 按项目筛选 | 多项目素材 | GET /api/assets?project_id=X | 200 | P1 |
| TC-API-ASSET-010 | 同步素材 | 文件系统有新文件 | POST /api/assets/sync | 200, 素材已同步 | P0 |
| TC-API-ASSET-020 | 上传素材到项目 | 项目已创建 | POST /api/assets/projects/{pid}/upload | 200, 素材已上传 | P0 |
| TC-API-ASSET-021 | 上传-项目不存在 | 无 | POST /api/assets/projects/99999/upload | 404 | P0 |
| TC-API-ASSET-022 | 上传-文件过大 | 项目已创建 | POST 超大文件 | 413 或 422 | P1 |
| TC-API-ASSET-023 | 上传-不支持的格式 | 项目已创建 | POST .exe 文件 | 422, 格式不支持 | P1 |
| TC-API-ASSET-030 | 获取素材详情 | 素材存在 | GET /api/assets/{id} | 200 | P0 |
| TC-API-ASSET-031 | 素材不存在 | 无 | GET /api/assets/99999 | 404 | P0 |
| TC-API-ASSET-040 | 获取素材文件 | 素材存在 | GET /api/assets/{id}/file | 200, 返回文件流 | P0 |
| TC-API-ASSET-041 | 文件不存在 | 素材记录存在但文件丢失 | GET /api/assets/{id}/file | 404 或 500 | P1 |
| TC-API-ASSET-050 | 上传素材内容 | 素材记录已创建 | POST /api/assets/{id}/upload | 200 | P1 |
| TC-API-ASSET-060 | 删除素材 | 素材存在 | DELETE /api/assets/{id} | 200/204 | P0 |
| TC-API-ASSET-061 | 删除不存在素材 | 无 | DELETE /api/assets/99999 | 404 | P0 |
| TC-API-ASSET-062 | 删除被引用素材 | 素材被分镜引用 | DELETE /api/assets/{id} | 409 或级联处理 | P1 |

```python
class TestAssetAPI:
    """素材管理 API 测试集"""

    @pytest.mark.asyncio
    async def test_tc_api_asset_020_upload(
        self, client: AsyncClient, project
    ):
        """TC-API-ASSET-020: 上传素材"""
        # 创建测试图片
        import io
        from PIL import Image
        img = Image.new("RGB", (100, 100), color="red")
        buffer = io.BytesIO()
        img.save(buffer, format="PNG")
        buffer.seek(0)

        response = await client.post(
            f"/api/assets/projects/{project.id}/upload",
            files={"file": ("test.png", buffer, "image/png")}
        )
        assert response.status_code == 200
        data = response.json()
        assert data["asset_type"] == "image"
        assert data["file_name"] == "test.png"
```
