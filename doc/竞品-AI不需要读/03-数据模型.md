# 03 数据模型

> 所有项目数据**没有数据库**，全部以"邻居目录 + JSON + PNG"组织。本节列出每个文件的关键字段、必填项、典型写入点。

## 1. 顶层

### 1.1 `project.json`

```json
{
  "id": "uuid",
  "name": "项目名",
  "version": 1,
  "createdAt": "2025-…",
  "updatedAt": "2025-…",
  "defaultProjectDirectory": "D:/.../projects",
  "settings": {
    "provider": "volc-ark",
    "imageModel": "doubao-seedream-...",
    "videoModel": "doubao-seedance-...",
    "textModel": "deepseek-v3"
  }
}
```

写入点：新建项目、修改项目设置。

  "cameraMovement": "跟拍",
  "shotType": "动态分镜",

  "lastFramePrompt": "中文：…",

- `001.mp4` / `001.wav`：可选视频 / 音频
- 缩略图：生成时实时由 `html-to-image` 导出

## 4. 任务中心

任务中心不落盘在项目里，而是渲染端用 zustand + idb 维护：

```ts
type Task = {
  id: string;
  type: "image"|"video"|"text"|"audio";
  status: "queued"|"running"|"success"|"failed"|"canceled";
  title: string;
  projectPath: string;
  shotId?: string;
  startedAt: number;
  finishedAt?: number;
  error?: { code: string; message: string };
  chunks?: string[];         // 流式 chunk
  result?: { path: string; meta?: any };
};
```

`idb` 用于"任务中心抽屉"在重启后能恢复最近 7 天的任务。

## 5. 全局配置（userData）

`userData/`（`app.getPath('userData')`）下：

```
userData/
├── .enc                       # safeStorage 加密的供应商凭证
├── claude-skills/             # 用户自定义 Claude Skill
├── plugins/                   # 用户安装的第三方插件
├── data/                      # storage:get-base-path 返回的位置
├── logs/                      # logger:get-dir 返回的位置
└── integrity-error.log        # 完整性错误日志
```

## 6. 供应商 / 模型 / 提示词模板

均为渲染端 zustand store（持久化到 `userData/data/*.json`）：

```ts
type Provider = {
  id: string;
  name: string;
  type: "openai-compatible" | "volc-ark" | "claude-agent" | "comfy";
  baseUrl?: string;
  apiKeyEncrypted?: string;  // 永远不返回明文
  region?: string;
  enabled: boolean;
  models: Model[];
};

type Model = {
  id: string;
  name: string;
  kind: "text"|"image"|"video"|"audio";
  params: Record<string, any>;
  defaultFor: "text"|"image"|"video"|"audio";
};

type PromptTemplate = {
  id: string;
  name: string;
  scope: "script" | "shot" | "character" | "scene" | "prop";
  body: string;        // 支持 {{role}}, {{context}} 等占位符
  builtin: boolean;
};
```

## 7. 媒体水印 `wlmjMediaMeta`

为方便追踪，PNG / MP4 / WAV 字节流末尾追加一个**自定义 chunk**：

```
[文件原有字节]
[WlmjMediaMeta Container]
  - magic    : "WL01"
  - kind     : "asset" | "oss-cache"
  - payload  : JSON 字符串
```

`mediaMeta:embed` / `mediaMeta:read` / `mediaMeta:read-by-kind` 三个 IPC 处理。

asset 元数据示例：

```json
{
  "kind": "asset",
  "assetType": "image|video|audio",
  "projectId": "…",
  "shotId": "…",
  "characterId": "…",
  "sceneId": "…",
  "propId": "…",
  "prompt": "…",
  "provider": "volc-ark",
  "model": "doubao-seedream-…",
  "createdAt": "…"
}
```

oss-cache 元数据示例：

```json
{
  "kind": "oss-cache",
  "fingerprint": "sha1",
  "provider": "zhongzhuan"|"oss-aliyun",
  "publicUrl": "…",
  "expiresAt": "…"
}
```
