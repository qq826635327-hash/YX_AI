# 10 首页 / 项目管理

## 1. 入口与路由

应用启动后默认进入 **`/home`**，由 `App.tsx` 中的 `<Home />` 组件渲染。

```
App.tsx
└─ <Routes>
   ├─ /home                    → <Home />          ← 应用入口
   ├─ /projects/:projectId     → <ProjectLayout>
   │    ├─ /                   → <ProjectOverview>
   │    ├─ /script             → <ScriptAndShots>
   │    ├─ /characters         → <Characters>
   │    ├─ /scenes             → <Scenes>
   │    ├─ /props              → <Props>
   │    ├─ /canvas             → <CanvasList>
   │    └─ /canvas/:canvasId   → <CanvasEditor>
   └─ /settings                → <Settings>
        ├─ /providers          → <ProviderSettings>
        ├─ /models             → <ModelSettings>
        ├─ /prompts            → <PromptTemplates>
        ├─ /comfyui            → <ComfySettings>
        ├─ /plugins            → <PluginSettings>
        └─ /general            → <GeneralSettings>
```

## 4. 项目管理

### 4.1 新建项目

UI 元素：

- 项目名输入框
- 项目目录选择（`dialog:open-directory` + `fs:validate-project-dir`）
- 模板下拉（空 / 教程示例）

提交后：

```
1. fs:mkdir 目录
2. fs:write-file project.json
3. fs:write-file 角色/角色.json { version, items: [] }
4. fs:write-file 场景/场景.json { version, items: [] }
5. fs:write-file 道具/道具.json { version, items: [] }
6. fs:write-file 剧集/剧集.json { version, items: [] }
7. fs:write-file 画布/canvas/.gitkeep
8. fs:write-file 生成/.gitkeep
9. directoryWatcher.startWatch(root)
10. navigate(/projects/:id/)
```

### 4.2 打开项目

- 选择目录 → 校验存在 `project.json` → 读出 → 跳转到对应项目主页
- "最近项目"列表来自 `userData/data/recent-projects.json`，按 mtime 倒序，最多 20 条

### 4.3 切换工作目录

- 命令面板（Ctrl+K）输入"切换工作目录"
- 修改 `app.getDefaultProjectDir()` 返回的默认目录

### 4.4 关闭 / 退出

- 单窗口：点 × 关闭 = `app:window-close`
- 退出：顶栏菜单 / 系统托盘"退出"
- `before-quit` → `pluginHost.shutdown()` + `claudeAgent.shutdown()`

## 5. 关键 IPC

- `app:get-default-project-dir` → 首次启动时用
- `fs:validate-project-dir` → 用户手填路径时立即校验
- `fs:read-file` / `fs:write-file` → project.json
- `dialog:open-directory` / `dialog:save-file`
- `directoryWatcher:start-watch` → 打开项目后开始监听
