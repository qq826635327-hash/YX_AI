# 11 项目总览

## 1. 路由

`/projects/:projectId/` → `<ProjectOverview />`

## 2. 布局

```
┌─────────────── 顶栏 ────────────────────────────────┐
│ 项目名 │ 任务中心 │ 缩放 │ 主题 │ 设置              │
├──────────┬────────────────────────┬───────────────────┤
│  左侧导航 │  统计卡片 + 任务列表   │  最近编辑资产     │
└──────────┴────────────────────────┴───────────────────┘
```

## 3. 统计卡片

- 总剧集数、总分镜数、已完成、失败、运行中
- 角色 / 场景 / 道具 数量
- 画布数量
- 本月生成次数

数据来源：扫描项目目录各 JSON 文件的 `items.length`，或读取 `剧集.json` 汇总。

## 4. 任务列表（最近 20 条）

按 `startedAt` 倒序，显示：

- 任务类型图标（生文 / 生图 / 生视频）
- 任务标题（如"分镜 001 首帧"）
- 状态徽标
- 时长

点击任务 → 跳到对应分镜 / 资产。

## 5. 最近编辑资产

按 `updatedAt` 倒序的角色 / 场景 / 道具 / 分镜混合列表，前 12 条。

## 6. 关键 IPC

- `fs:read-file` × N
- `fs:stat`
- `directoryWatcher:onFilesChanged` → 局部刷新
- 不调用任何出站请求，纯本地数据汇总

## 7. 性能

- 不在前端解析所有图片缩略图
- 缩略图由 `image-host` bundle 按需懒加载（`html-to-image` 或浏览器原生）
