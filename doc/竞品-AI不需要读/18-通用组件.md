# 18 通用组件

> 几乎所有页面都会复用以下组件，集中定义在 `src/components/`。

## 1. 资产卡片 (`<AssetCard />`)

- props: `asset, kind, selected, onSelect, onOpen, onAction`
- 显示缩略图、名称、状态徽标、操作菜单
- 拖动支持：跨列表移动

## 2. 编辑对话框 (`<EditDialog />`)

- 通用模态：表单字段以 schema 形式描述
- 支持：input / textarea / select / 多选 / 富文本 / Markdown / 颜色 / 滑块
- 自动保存：失焦后写盘 + 触发 directoryWatcher

## 3. 生成对话框 (`<GenerationDialog />`)

字段：

- 模型（按 kind 过滤）
- 风格 / 比例 / 数量
- 提示词（带"提示词助手"按钮 + 模板下拉）
- 参考图（拖入 / 选）
- 高级参数（折叠）
- 底部：[生成] [取消]

提交后：
1. 派任务到 useTaskStore
2. 立即关闭（也可以选"生成时保持打开"）
3. 进度在任务中心抽屉里

## 4. 灯箱 (`<Lightbox />`)

- 全屏查看图片 / 视频
- 左右键切换、滚轮缩放、双击重置
- 工具栏：下载、复制、设为形象、加入参考图、生成相似图

## 5. 素材画廊 (`<MaterialGallery />`)

- 横跨角色 / 场景 / 道具 / 分镜结果的统一浏览
- 标签筛选 + 搜索 + 按类型过滤
- 拖入到任意"参考图"槽位

## 6. 提示词模板选择器 (`<PromptTemplatePicker />`)

- 列出当前 scope 的所有模板
- 内置 + 用户自定义
- 点选即填入

## 7. Markdown 渲染器

用 `react-markdown` 包装，支持：

- `代码块`（带语法高亮）
- `表格`
- `引用`
- `<!--SHOT:id-->` / `<!--SCENE:id-->` 注释高亮（点击跳转）

## 8. 命令面板 (`<CommandPalette />`)

- Ctrl+K 唤起
- 全局搜索 + 命令触发
- 模糊匹配

## 9. 通用确认对话框

- 普通 / 危险（红色按钮）
- 支持"记住选择"复选框

## 10. Toast

- 成功 / 失败 / 信息
- 自动消失 / 可堆叠
