# 23 设置 · 主题与系统

## 1. 入口

`/settings/general`

## 2. 主题

- 浅色 / 深色 / 跟随系统
- 主色（蓝 / 紫 / 绿）
- 圆角 / 字体 / 紧凑度

## 3. 缩放

- 全局缩放（`webFrame.setZoomFactor`）
- 顶栏快捷加减按钮

## 4. 通用

- 工作目录（默认项目目录）
- 启动行为（最大化 / 记住尺寸 / 默认进入最近项目）
- 文件监听开关

## 5. 安全面板

- 显示 app.asar 完整性状态
- 完整性错误日志查看
- 重新校验
- "忽略完整性校验启动"（仅高级用户，明示风险）

## 6. 日志

- 主进程日志目录（`logger:get-dir`）
- 渲染端日志（实时）
- 项目级日志（每个项目 `日志/`）

## 7. 版本

- 应用版本号
- Electron / Chromium / Node 版本
- 检查更新（若有 update server）
- 关于 / 许可 / 第三方声明

## 8. 关键 IPC

- `app:get-version` / `app:is-packaged` / `app:get-arch`
- `fingerprint:get` / `diag`
- `logger:get-dir` / `open-dir`
- `system:get-info`
- `zoom:set-factor` / `get-factor`
