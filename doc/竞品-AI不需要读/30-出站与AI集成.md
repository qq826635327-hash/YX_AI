# 30 出站与 AI 集成

> 「1人漫剧」所有 AI 调用的统一入口都是 `outbound:request`。这是 AI 复刻必须搞定的核心层。

## 1. 抽象目标

- **屏蔽供应商差异**：OpenAI 兼容 / 火山方舟 / 千问 / DeepSeek / Claude / ComfyUI 全部走同一接口
- **流式 chunk 可控**：`outbound:set-chunk-broadcast` 决定是否推到 renderer
- **可取消**：返回 `token`，调 `outbound:cancel(token)` abort
- **可重试**：自动重试 1 次，失败再走用户主动重试
- **超时可配**：`outbound:set-defaults({ timeoutMs })`
- **密钥安全**：API key 在主进程 secret 内存中，不在 renderer 出现

## 2. `outbound:request` 入参

```ts
type OutboundRequest = {
  providerId: string;                    // 来自供应商配置
  model: string;
  kind: "text" | "image" | "video" | "audio";
  // text
  messages?: Array<{ role: "system"|"user"|"assistant"; content: string }>;
  prompt?: string;
  // image
  images?: string[];                     // base64 / URL
  // common
  params?: Record<string, any>;          // temperature / size / style / ...
  stream?: boolean;
  // 透传
  meta?: Record<string, any>;            // 透传到 metadata / 提示词变量
};
```

## 3. 出参

```ts
type OutboundResponse = {
  ok: boolean;
  data?: any;                            // 文本 / 图片 URL / 视频 URL
  error?: { code: string; message: string };
  token?: string;                        // 取消用
};
```

## 4. 流式事件

```
electron.ipcRenderer.on("outbound:event", (e, payload) => {
  // payload = { token, type: "chunk"|"done"|"error", payload: any }
});
```

- `chunk`：文本 delta / 图片 partial URL / 进度数字
- `done`：成功结束
- `error`：异常

## 5. 主进程实现要点

```ts
async function outboundRequest(req: OutboundRequest) {
  const provider = providers.get(req.providerId);
  if (!provider) throw { code: "AuthError", message: "unknown provider" };

  const token = uuid();
  const controller = new AbortController();
  cancelTokens.set(token, controller);

  const secrets = secretStore.get(providerId);
  try {
    const stream = adapter.call(provider, secrets, req, controller.signal);
    if (req.stream) {
      for await (const chunk of stream) {
        webContents.send("outbound:event", { token, type: "chunk", payload: chunk });
      }
    } else {
      const data = await once(stream);
    }
    webContents.send("outbound:event", { token, type: "done" });
    return { ok: true, data, token };
  } catch (e) {
    webContents.send("outbound:event", { token, type: "error", payload: e });
    return { ok: false, error: e, token };
  } finally {
    cancelTokens.delete(token);
  }
}
```

## 6. 各 provider 适配器（推断）

// services/adapters/openai-compatible.ts
async function* callOpenAICompatible(provider, secrets, req) {
  const r = await fetch(provider.baseUrl + "/chat/completions", {
    method: "POST",
    headers: { Authorization: `Bearer ${secrets.apiKey}` },
    body: JSON.stringify(toOpenAIRequest(req)),
    signal,
  });
  if (!r.ok) throw mapError(r);
  if (req.stream) {
    for await (const chunk of parseSSE(r)) {
      yield chunk.choices[0]?.delta?.content;
    }
  } else {
    yield await r.json();
  }
}

## 8. 复刻提示

- 写 adapter 时把 **"鉴权 → 请求拼装 → SSE 解析 → 错误归一"** 拆成 4 步，每步可单测
- 火山方舟图像返回 URL，**需要立即下载到本地**（主进程 fetch → 二进制 → fs:write-file-binary），不要保存远程 URL
- ComfyUI 流式靠 WebSocket `client_id` 过滤
