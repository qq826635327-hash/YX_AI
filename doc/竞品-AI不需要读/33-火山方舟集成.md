# 33 火山方舟集成

## 1. 用途

- 文本模型（豆包）
- 图像模型（豆包图像 / Seedream）
- 视频模型（豆包 / Seedance）
- 数字人 / 真人素材库（`volc:list-assets`）

## 2. 凭证

- 走 `safeStorage` 加解密
- 保存：`volc:save-credentials({ accessKeyId, secretAccessKey, region? })`
- 默认 region：`cn-beijing`
- Linux 无 `libsecret` 时拒绝保存

## 3. 测通

- `testCredentials(input)` → `ListAssets` pageSize=1
- 错误码：
  - `InvalidAccessKey` → AK 错
  - `SignatureDoesNotMatch` → SK 错 / 系统时间不对
  - status=0 → 网络问题

## 4. 资产列表

```ts
window.electronAPI.volc.listAssets({
  groupType?: "AIGC" | "LivenessFace",
  pageNumber: number,   // 1-based
  pageSize: number      // 最大 100
})
// → { ok, items: [{ Key, URL, LastModified, FileSize, ... }], total }
```

- `URL` 是 12h 签名 URL
- 渲染端用 `useVolcAssetsStore` 缓存 12h TTL

## 5. 缩略图下载

- 浏览器直接 fetch 缩略图会被 CORS 拦
- 用 `volc:fetchAssetBytes(url)` → 主进程 net.fetch → ArrayBuffer
- 限制：URL `https://` 开头、长度 ≤ 4096、超时 30s、字节 ≤ 16MB

## 6. 显示明文凭证

- `revealCredentials` → 弹出"眼睛"按钮时调一次
- 返回 AK + SK + region
- **不**打日志、**不**上报

## 7. 复刻提示

- 火山方舟的 image / video 端点遵循 OpenAI 兼容或方舟自研格式，复刻时先看 `docs/volc/...`（外部文档）
- 资产缩略图必须走主进程下载，**不能**在浏览器 fetch
