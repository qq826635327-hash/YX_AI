## 风险 / 决策点

1. **数据库**：复刻可加 SQLite 做"项目元信息索引"，但项目数据**仍**落 JSON 目录，不要破坏文件结构
2. **Tailwind vs CSS Modules**：产品用 Tailwind（推断），复刻可直接用
3. **国际化**：当前产品只有中文，复刻可考虑加入英文但不强求
4. **更新机制**：v0.1.58 没有 update server，靠用户重装；可加 electron-updater
5. **云协作**：原产品无云端，复刻可加"项目同步到 S3/OSS"作为高级特性
6. **视频剪辑**：当前是"按分镜生成短视频"，复刻不一定要做长视频剪辑

## 第一份可以交付的"可运行"版本
