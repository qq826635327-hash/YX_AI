## 任务 1：start.bat 启动失败修复（最终方案）

### 目标
修复 start.bat 经常无法启动前后端的问题

### 结果
经过 4 轮迭代修复，最终方案：核心逻辑从 cmd bat 迁移到 PowerShell，bat 只做入口调用。踩坑：①cmd /c 引号嵌套导致进程无法启动；②chcp 65001 导致 cmd 把中文路径拆开（影序AI → 影序 + AI，AI 被当命令执行）；③PowerShell .ps1 文件用 UTF-8 保存时，Windows PowerShell 默认用 GBK 读取导致中文全乱码；④bat 执行完直接关闭窗口看不到结果。最终方案：bat 用 pushd+相对路径调 ps1（不设 chcp），ps1 输出全用英文，成功/失败都 pause 等按键

### 修改文件
- start.bat（精简为入口：pushd + start powershell + popd）
- scripts\restart.ps1（新建：PowerShell 核心逻辑，全英文输出）
